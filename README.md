## Sobre
O projeto é um curso que ensina como utilizar a IDE Qt Creator e o Qt SDK para criar pequenos aplicativos desktop.

## Conteúdo do curso
* Componentes Básicos
* Entrada de Dados
* Seleção de Dados
* Manipulando Tabelas
* Utilizando Imagens
* Subformulários
* Menus e Toolbars
* Caixas Texto e Arquivos
* Layout de Formulário
